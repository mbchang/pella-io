Harmonize

Harmonize is an end-to-end, automatic music augmentation tool. Provided with a melody or even just a single note, Harmonize will create a full musical arrangement of the piece for four different vocal ranges. At the core of our project is a musical generative model, which will accept either written notes or a raw audio file of a simple waveform. The result is both a MIDI file with the melody line and harmonies and written sheet music for performance by human singers, a capella style!
